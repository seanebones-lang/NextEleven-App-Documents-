## Support Process

### Step 1: Check Documentation

Before contacting support:
1. Check the FAQ document
2. Review Onboarding and Troubleshooting guide
3. Check app Settings for help options
4. Review relevant documentation in GitHub repository

### Step 2: Contact Support

When contacting support:
1. **Email:** info@mothership-ai.com
2. **Subject:** Clear, descriptive subject line
3. **Details:** Include:
   - Your issue or question
   - Steps to reproduce (if applicable)
   - Device model and iOS version
   - App version
   - Screenshots (if helpful)
   - What you've already tried

### Step 3: Wait for Response

- **Standard:** 24-48 hours
- **Priority:** 12-24 hours (subscribers)
- **Emergency:** Within 24 hours

### Step 4: Follow Up

- If no response after expected time, send a polite follow-up
- Include original message for context
- Be patient - we're a small team
