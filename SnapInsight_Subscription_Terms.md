## Subscription Management

### How to Upgrade
1. Open SnapInsight
2. Settings → Upgrade to Pro/Dev
3. Choose tier
4. Complete Apple payment
5. Add your xAI API key (Settings → API Configuration)

### How to Downgrade
1. iPhone Settings → Subscriptions → SnapInsight
2. Select lower tier or cancel
3. Changes apply at next renewal

### How to Restore Purchases
If you subscribed on another device:
1. Open SnapInsight
2. Settings → Restore Purchases
3. Sign in with same Apple ID
4. Subscription status syncs

### Family Sharing (Dev Tier Only)
**Apple Family Sharing:**
- Not supported for Pro tier

**API Key Sharing (Dev):**
1. Dev subscriber gets API key

3. Family members install SnapInsight (free)
4. Add shared key in Settings
