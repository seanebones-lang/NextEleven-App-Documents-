## Common Issues & Solutions

### 1. Subscription Not Recognized

**Symptoms:**
- Pro/Dev features locked despite subscription
- "Upgrade to Pro" prompts when subscribed

**Solutions:**
1. Restore Purchases: Settings → Restore Purchases
2. Check Apple subscription: Settings → [Your Name] → Subscriptions
3. Sign in with correct Apple ID
4. Restart app
5. Contact us if still broken (include receipt screenshot)

### 2. API Key Validation Fails

**Symptoms:**
- "Invalid API key" error
- Key validation never completes
- 401 Unauthorized from xAI

**Solutions:**
1. Check key at x.ai/console (ensure active)
2. Copy key again (no extra spaces)
3. Verify key starts with "xai-" and is 40+ characters
4. Ensure xAI account has available credit
5. Try removing and re-adding key

### 3. Analyses Not Working

**Symptoms:**
- Loading spinner forever
- "Network error" messages
- Results not appearing

**Solutions:**
1. Check internet connection
2. Verify API key (Pro/Dev users)
3. Check xAI status: https://status.x.ai
4. Try smaller image (compress large photos)
5. Restart app
6. Contact us with error screenshot

### 4. Share to X Failing

**Symptoms:**
- "Authorization failed" when sharing
- Post doesn't appear on X

**Solutions:**
1. Disconnect and reconnect X: Settings → X Integration
2. Check X account status (not suspended)
3. Review post content (no policy violations)
4. Try shorter caption
5. Check X API status
