## Common Issues and Solutions

### Scanning Issues

**Issue: Receipts won't scan or OCR fails**

Solutions:
1. Ensure good lighting (natural light works best)
2. Clean camera lens
3. Flatten receipt to remove wrinkles
4. Hold device steady for 2-3 seconds
5. Update app to latest version
6. Restart app

**Issue: Scanned data is inaccurate**

Solutions:
1. Re-scan with better lighting/angle
2. Manually correct extracted data
3. Use high-quality receipts when possible
4. Ensure receipt text is clearly legible
5. Report persistent issues to support

### Subscription Issues

**Issue: Premium features not unlocked after purchase**

Solutions:
1. Check internet connection
2. Restart app completely
3. Go to Settings → Restore Purchases
4. Verify correct Apple ID is signed in
5. Wait 5-10 minutes and restart again
6. Contact support if still not working

**Issue: Can't cancel subscription**

Solutions:
1. Open iOS Settings → [Your Name] → Subscriptions
2. Find Scanner App by NextEleven
3. Tap Cancel Subscription
4. Or visit: https://apps.apple.com/account/subscriptions
5. For refunds, contact Apple at reportaproblem.apple.com

### Sync Issues

**Issue: iCloud sync not working**

Solutions:
1. Verify iCloud Drive is enabled (Settings → iCloud)
2. Ensure sufficient iCloud storage available
3. Check internet connection
4. Toggle iCloud sync off/on in app Settings
5. Sign out and back into iCloud
6. Restart device

**Issue: Data not syncing between devices**

Solutions:
1. Ensure both devices have iCloud sync enabled
2. Check same Apple ID on both devices
3. Verify internet connection on both devices
4. Force sync: pull down to refresh in app
5. Check for app updates on both devices

### Performance Issues

**Issue: App is slow or laggy**

Solutions:
1. Close other apps to free up memory
2. Restart Scanner App
3. Restart device
4. Check available storage space
5. Update to latest iOS version
6. Update to latest app version
7. Consider archiving old receipts

**Issue: App crashes frequently**

Solutions:
1. Update to latest app version
2. Update to latest iOS version
3. Restart device
4. Check available storage (need 1GB+ free)
5. Reinstall app (backup data first)
6. Contact support with crash details

## Data and Privacy Support

### Data Recovery

**Lost data after app reinstall:**
- Enable iCloud sync and sign in with same Apple ID
- Data should automatically restore
- If not, check iCloud settings and storage

**Accidentally deleted receipts:**
- Currently no undo feature
- Check iCloud backups if sync enabled
- Consider regular data exports (premium feature)

**Device upgrade or replacement:**
1. Ensure iCloud sync enabled on old device
2. Install app on new device
3. Sign in with same Apple ID
4. Enable iCloud sync
5. Or use Settings → Restore Purchases

### Privacy Requests

**Access your data:**
- All data is stored locally on your device
- Access anytime through the app
- Export as PDF (premium) or manually

**Delete your data:**
- Delete receipts individually or in bulk
- Uninstall app to remove all local data
- Disable iCloud sync and delete from iCloud

**Data portability:**
- Export receipts as PDF (premium)
- Export data as CSV (premium, coming soon)
- Manual export via screenshots

## Refund Policy

### Requesting Refunds

All purchases are processed by Apple. We cannot issue refunds directly.

**To request a refund:**
1. Visit https://reportaproblem.apple.com
2. Sign in with your Apple ID
3. Find Scanner App by NextEleven purchase
4. Select "Request a Refund"
5. Choose reason and submit

**Apple's decision is final.** We have no control over refund approvals.

### When Refunds May Be Approved

Apple typically grants refunds for:
- Accidental purchases
- Technical issues preventing use
- Unauthorized charges
- Subscription charged after cancellation

### When Refunds May Be Denied

Apple typically denies refunds for:
- Change of mind
- Forgetting to cancel trial
- Not using the app
- Feature expectations not met
