## What We Support

### ✅ Supported Issues

- **Technical Problems:** App crashes, freezes, errors
- **Feature Questions:** How to use app features
- **Bug Reports:** Issues with calculations, scanning, or data
- **Privacy Concerns:** Data handling and privacy questions
- **Account Issues:** Subscription and billing questions
- **Compatibility:** iOS version and device compatibility
- **Data Recovery:** Help recovering lost shopping history (if possible)

### ❌ Not Supported

- **Third-Party Services:** Issues with xAI API, product lookup services, or tax services
- **Device Issues:** Hardware problems, iOS bugs, or device-specific issues
- **Network Problems:** Internet connectivity or network configuration issues
- **Custom Development:** Feature customization or custom integrations
- **Legal Advice:** We cannot provide legal advice

## How to Get Help

### Before Contacting Support

1. **Check the FAQ:** Many common questions are answered in our FAQ
2. **Review Troubleshooting Guide:** Common issues and solutions are documented
3. **Update the App:** Ensure you're using the latest version
4. **Restart the App:** Close and reopen the app
5. **Restart Your Device:** Sometimes a device restart resolves issues

### When Contacting Support

Please include the following information:

1. **Your Issue:** Clear description of the problem
2. **Steps to Reproduce:** What you were doing when the issue occurred
3. **Device Information:** iPhone model, iOS version
4. **App Version:** Version number from Settings
5. **Screenshots:** If applicable, include screenshots or screen recordings
6. **Error Messages:** Any error messages you received

### Example Support Request

```
Subject: [Shopper] Issue with tax calculation

Device: iPhone 15 Pro, iOS 17.2
App Version: 1.0
Issue: Tax calculation showing incorrect amount for California
Steps to Reproduce:
1. Added item for $100
2. Selected California as state
3. Tax shows $8.25 instead of expected $7.25

[Screenshot attached]
```

## Bug Reports

We appreciate bug reports! When reporting bugs:

- **Be Specific:** Describe exactly what happened
- **Include Details:** Device, iOS version, app version
- **Reproduction Steps:** Steps to reproduce the issue
- **Expected vs Actual:** What you expected vs what actually happened
- **Screenshots/Video:** Visual evidence is very helpful

## Feature Requests

We welcome feature requests! When suggesting features:

- **Describe the Feature:** What you'd like to see
- **Use Case:** How would you use this feature?
- **Priority:** Is this a must-have or nice-to-have?

**Note:** We cannot guarantee that all feature requests will be implemented, but we review all suggestions and consider them for future updates.
