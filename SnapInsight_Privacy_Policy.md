## Our Commitment to Privacy

SnapInsight is built with privacy as a core principle. We believe your photos, videos, and AI analyses are yours alone. This policy explains exactly what data we collect (very little), how we use it, and your rights.

**Privacy-First Design:**
- No cloud storage of your media
- No tracking or advertising
- No data selling or monetization
- No account creation required
- 100% local processing by default
- User-controlled third-party integration

---

## What Data We Collect

### 1. Subscription Information (Collected by Apple)

**What We Collect:**
- Subscription tier (Free, Pro, Dev)
- Subscription status (active, expired, canceled)
- Purchase receipts (via Apple StoreKit)

**How We Collect It:**
- Apple StoreKit 2 framework
- On-device verification only
- No data transmitted to our servers

**How We Use It:**
- Unlock premium features based on subscription
- Verify subscription status locally
- Restore purchases across devices

**Data Retention:**
- Stored locally on your device
- Synced via iCloud (Apple's system)
- Deleted when app is uninstalled

### 2. App Settings and Preferences

**What We Store:**
- Selected agents and configurations
- App preferences (haptics, sounds, theme)
- Analysis history (photos/videos + AI results)
- Custom agent chains (if created)

**Where It's Stored:**
- Locally on your device (SwiftData)
- Optionally synced via iCloud (if enabled in iOS Settings)

**How We Use It:**
- Provide personalized experience
- Maintain analysis history
- Restore settings after app updates

**Data Retention:**
- Stored until you delete it (swipe to delete in History)
- Clear all history via Settings → Clear History
- Deleted completely when app is uninstalled

### 3. API Keys (User-Provided)

**What We Store:**
- Your xAI Grok API key (Pro/Dev tiers only)

**Where It's Stored:**
- iOS Keychain (encrypted by Apple, not accessible to us)
- Never transmitted to SnapInsight servers
- Never logged or analyzed

**How We Use It:**
- Authenticate your API requests to xAI
- Loaded only when you initiate analysis
- Sent directly from your device to api.x.ai

**Data Retention:**
- Stored in Keychain until you remove it
- Settings → API Configuration → Remove Key
- Deleted when app is uninstalled

### 4. Usage Analytics (Optional, Opt-In)

**What We Collect (if you enable Apple Analytics):**
- App crashes and performance issues
- Feature usage (e.g., which agents are most popular)
- Device type and iOS version

**How We Collect It:**
- Apple's built-in App Analytics (if enabled in iOS Settings → Privacy)
- Anonymized and aggregated by Apple

**How We Use It:**
- Fix bugs and improve stability
- Understand which features to prioritize
- Optimize performance

**How to Opt-Out:**
- iOS Settings → Privacy & Security → Analytics & Improvements → Share iPhone Analytics (toggle OFF)

**Data Retention:**
- Handled by Apple's privacy policy

## Third-Party Data Sharing

### xAI (x.ai) - AI Processing

**When Data is Shared:**
- When you use analysis features (Pro/Dev tiers with your own API key)
- When Free tier users analyze media (using app-provided key)

**What Data is Shared:**
- Photos or videos you choose to analyze
- Your API key (sent directly from your device, not through us)
- AI prompts (generated by selected agents)

**How Data Flows:**
```
Your Device → [Your API Key + Media] → xAI API (api.x.ai) → Results → Your Device
                                       ↑
                           SnapInsight DOES NOT intercept
```

**Their Purpose:**
- Process media through Grok AI models
- Generate analysis results

**Their Privacy Policy:**
- https://x.ai/legal/privacy-policy

**Your Control:**
- Free tier: Use up to 3 analyses/day (our key)
- Pro/Dev: Provide your own key = full control
- Don't analyze = no data shared

**Data Retention by xAI:**
- Subject to xAI's privacy policy
- We recommend reviewing their terms

### X/Twitter (Optional, User-Initiated)

**When Data is Shared:**
- When you explicitly tap "Share on X" button
- OAuth 2.0 authentication (you authorize access)

**What Data is Shared:**
- Photo or video you choose to share
- AI analysis caption you approve
- Your X account credentials (OAuth token)

**How It Works:**
- You tap "Share on X"
- Authenticate with X (standard OAuth)
- Review post before publishing
- You confirm share → data sent to X

**Their Privacy Policy:**
- https://x.com/en/privacy

**Your Control:**
- Completely optional (never required)
- Revoke access: X Settings → Apps and Sessions
- Disconnect: SnapInsight Settings → X Integration → Disconnect

**Data Retention by X:**
- Subject to X/Twitter's privacy policy

### Apple Services

**StoreKit (Subscriptions):**
- Handles all payment processing
- Apple's privacy policy applies
- https://www.apple.com/legal/privacy/

**iCloud (Optional Sync):**
- If enabled in iOS Settings → iCloud
- Syncs app settings, history, API key
- Apple's privacy policy applies
- Disable: iOS Settings → [Your Name] → iCloud → SnapInsight (toggle OFF)

## Data Security

### On-Device Security

**Encryption:**
- All local data encrypted by iOS
- API keys in Keychain (AES-256 encryption)
- SwiftData storage (iOS encryption)

**App Sandboxing:**
- iOS app sandbox prevents unauthorized access
- No other apps can access SnapInsight data

**Code Signing:**
- App signed by Apple Developer certificate
- Verified integrity on every launch

### Data in Transit

**HTTPS Only:**
- All API calls use HTTPS (TLS 1.3+)
- Certificate pinning for xAI API
- No unencrypted data transmission

**API Authentication:**
- Bearer token authentication (industry standard)
- API keys never logged or stored on servers

### No Cloud Storage

**Your Media:**
- Never uploaded to SnapInsight servers
- Never stored in SnapInsight cloud
- Stays on your device (or iCloud if you enable)

**Analysis Results:**
- Stored locally on your device
- Optionally synced via iCloud (your choice)
- Never on SnapInsight servers

## Your Privacy Rights

### Access Your Data

**View in App:**
- All analysis history: History tab
- Settings and preferences: Settings tab
- API key status: Settings → API Configuration

**Export Your Data:**
- Individual analyses: Share button
- Screenshots or copy text from results

### Delete Your Data

**Individual Items:**
- Swipe left on any analysis in History → Delete

**Clear All History:**
- Settings → Clear History → Confirm

**Remove API Key:**
- Settings → API Configuration → Remove Key

**Delete Everything:**
- Uninstall SnapInsight from your device
- All local data permanently deleted

### Opt-Out of Analytics

**Apple Analytics:**
- iOS Settings → Privacy & Security → Analytics & Improvements → Share iPhone Analytics (toggle OFF)

**Stop Third-Party Sharing:**
- Don't use analysis features (no xAI sharing)
- Don't tap "Share on X" (no X sharing)
- Remove API key (stops Pro/Dev features)
