- How to use features

### 2.2 What We Don't Support

We do not provide support for:

- Third-party API issues (xAI Grok API problems - contact xAI support)

- Issues with unsupported iOS versions (below iOS 19.0)

## 4. How to Get Support

### 4.1 Before Contacting Support

1. **Check the FAQ** (Settings > FAQ)
2. **Review Troubleshooting Guide** (Settings > Troubleshooting)
3. **Check for App Updates** (App Store)
4. **Restart the App** (close and reopen)
5. **Restart Your Device** (if issues persist)

### 4.2 When Contacting Support

Please include:

**For Technical Issues:**
- Device model (e.g., iPhone 15 Pro)
- iOS version (Settings > General > About)
- App version (shown in Settings)
- Detailed description of the issue
- Steps to reproduce the problem
- Screenshots or screen recordings (if applicable)
- Error messages (if any)

**For Subscription Issues:**
- Apple ID email
- Subscription type (Monthly/Yearly)
- Purchase date
- Transaction ID (if available)
- Description of the issue

**For Feature Requests:**
- Detailed description of the feature
- Use case and benefits
- Any relevant examples

## 6. Bug Reports

### 6.1 Reporting Bugs

When reporting bugs, please provide:

1. **Device Information**
   - Device model
   - iOS version
   - App version

2. **Issue Description**
   - What you were doing when the bug occurred
   - What you expected to happen
   - What actually happened
   - Steps to reproduce

3. **Evidence**
   - Screenshots
   - Screen recordings
   - Error messages
   - Crash logs (if available)

4. **Frequency**
   - Does it happen every time?
   - When did it first occur?
   - Any patterns you've noticed?

### 6.2 Bug Fix Process

1. **Acknowledgment:** We'll confirm receipt within 5 business days
2. **Investigation:** We'll investigate and reproduce the issue
3. **Fix Development:** We'll develop a fix
4. **Testing:** We'll test the fix thoroughly
5. **Release:** We'll release the fix in the next app update

**Timeline:** Critical bugs are typically fixed within 1-2 weeks. Non-critical bugs may take longer depending on severity and resources.

We welcome feature requests! Please email us at info@mothership-ai.com with:

## 8. Refunds and Cancellations

### 8.1 Subscription Cancellations

You can cancel your subscription at any time:
- **iOS Settings:** Settings > [Your Name] > Subscriptions > Zombie Parrot
- **Cancellation takes effect** at the end of the current billing period
- **No refunds** for the current period (you retain access until period ends)

### 8.2 Refund Requests

Refunds are handled by Apple according to their policies:
- **Contact Apple Support** directly for refund requests
- **We cannot process refunds** directly
- **Refund eligibility** is determined by Apple

### 8.3 Subscription Issues

If you experience subscription issues:
1. Check your Apple ID subscription status
2. Verify payment method is valid
3. Try restoring purchases (Settings > Restore Purchases)
4. Contact us at info@mothership-ai.com if issues persist
