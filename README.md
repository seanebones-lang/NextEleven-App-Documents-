## App Store Connect Setup

### For Custom EULA (Recommended)

1. Go to App Store Connect
2. Navigate to your app â†’ App Information
3. Scroll to "End User License Agreement (EULA)"
4. Select "Use Custom EULA"
5. Upload or paste the EULA content from [EULA.md](./EULA.md)

### For Standard Apple EULA

1. Include the EULA link in your App Description:
   ```
   Terms of Use: https://github.com/seanebones-lang/NextEleven-App-Documents-/blob/main/EULA.md
   ```

2. Ensure the link is functional and accessible
