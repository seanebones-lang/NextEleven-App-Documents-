## API Key Setup (Optional)

### Using Default Key

- **No Setup Required:** App includes a default API key
- **Ready to Use:** Start chatting immediately
- **Limitations:** Subject to rate limits

### Using Your Own Key

**Why Use Your Own Key?**
- Avoid rate limits
- Control your API usage
- More reliable service

**How to Set Up:**
1. Get API key from https://x.ai/api
2. Go to Settings → API Configuration
3. Tap "Set Custom"
4. Enter your API key (starts with "xai-")
5. Tap "Save API Key"
6. Key is validated automatically

**How to Remove:**
1. Settings → API Configuration
2. Tap "Use Default Key"
3. Custom key is removed

**Troubleshooting:**
- **Invalid Key:** Make sure it starts with "xai-"
- **Validation Failed:** Check internet connection
- **Key Not Working:** Verify key is active at x.ai

## Common Issues and Solutions

### App Won't Open

**Symptoms:**
- App crashes on launch
- App won't start
- Black screen

**Solutions:**
1. Force close the app and reopen
2. Restart your device
3. Update iOS to latest version
4. Delete and reinstall app (data is stored locally)
5. Check available storage space
6. Contact support: info@mothership-ai.com

### Messages Not Sending

**Symptoms:**
- Messages stuck sending
- Error messages
- No response from AI

**Solutions:**
1. **Check Internet:** Ensure you have internet connection
2. **Check Subscription:** Verify subscription is active
3. **Wait and Retry:** May be rate limited - wait a few minutes
4. **Check API Key:** If using custom key, verify it's valid
5. **Restart App:** Force close and reopen
6. **Contact Support:** info@mothership-ai.com

### Rate Limiting Errors

**Symptoms:**
- "Rate limited" error
- "Too many requests" message
- Temporary blocking

**Solutions:**
1. **Wait:** Usually resolves in 1-5 minutes
2. **Use Your Own Key:** Set up custom API key in Settings
3. **Reduce Frequency:** Wait between messages
4. **Check Status:** xAI API may be experiencing issues

### Voice Not Working

**Symptoms:**
- Microphone not responding
- Voice not transcribed
- No audio output

**Solutions:**
1. **Grant Permissions:** Settings → Ultimate Life Coach → Microphone
2. **Enable Voice:** Settings in app → Voice Input enabled
3. **Real Device:** Voice works better on real device vs. simulator
4. **Restart App:** Force close and reopen
5. **Check Settings:** Verify voice toggles are enabled

### Photos Not Uploading

**Symptoms:**
- Photo picker not opening
- Photos not sending
- Analysis not working

**Solutions:**
1. **Grant Permission:** Settings → Ultimate Life Coach → Photos
2. **Check Size:** App compresses automatically, but very large photos may fail
3. **Try Different Photo:** Some formats may not work
4. **Check Internet:** Required for photo analysis
5. **Restart App:** Force close and reopen

### Subscription Issues

**Symptoms:**
- Can't access premium features
- Subscription not recognized
- Payment problems

**Solutions:**
1. **Check Status:** App Store → Subscriptions
2. **Restore Purchases:** App Store → Restore Purchases
3. **Contact Apple:** For payment issues, contact Apple Support
4. **Contact Us:** info@mothership-ai.com for app-specific issues

### Data Loss Concerns

**Symptoms:**
- Worried about losing data
- Want to backup
- Device replacement

**Solutions:**
1. **Export Regularly:** Settings → Export Chat History
2. **Local Storage:** Data is on your device
3. **Backup:** Export before device changes
4. **iCloud:** iOS may backup if enabled (check Settings)

### Slow Performance

**Symptoms:**
- App is slow
- Laggy responses
- Freezing

**Solutions:**
1. **Close Other Apps:** Free up device memory
2. **Restart Device:** Clear temporary files
3. **Update App:** Get latest version from App Store
4. **Check Storage:** Ensure adequate free space
5. **Long Conversations:** Very long histories may slow down

### API Key Validation Failed

**Symptoms:**
- Custom key not working
- Validation error
- Key rejected

**Solutions:**
1. **Check Format:** Must start with "xai-"
2. **Verify Key:** Check key is active at x.ai
3. **Check Internet:** Required for validation
4. **Try Again:** Network issues may cause temporary failures
5. **Use Default:** Fall back to default key if needed
