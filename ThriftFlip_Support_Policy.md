## How to Get the Best Support

### Before Contacting Us

**Try These First:**

1. **Check FAQ**
   - Settings > FAQ
   - Most common issues covered

2. **Check User Guide**
   - Settings > User Guide
   - Step-by-step instructions

3. **Restart App**
   - Force quit app
   - Reopen
   - Often fixes issues

4. **Update App**
   - App Store > Updates
   - Latest version has bug fixes

5. **Check Internet**
   - Verify connection works
   - Try different network

### When Emailing Support

**Include This Information:**

**Required:**
- iOS version (Settings > General > About)
- ThriftFlip version (Settings > About)
- Description of issue
- Steps to reproduce

**Helpful:**
- Screenshots or screen recording
- Error messages
- When issue started
- Debug log (Settings > Grok AI > Copy Debug Log)

**Email Template:**

```
Subject: ThriftFlip Support - [Brief Issue]

iOS Version: [e.g., 18.2]
ThriftFlip Version: [e.g., 2.0]
Device: [e.g., iPhone 15 Pro]

Issue Description:
[Describe what's happening]

Steps to Reproduce:
1. [First step]
2. [Second step]
3. [Error occurs]

Expected Behavior:
[What should happen]

Actual Behavior:
[What actually happens]

Screenshots: [Attached]
Debug Log: [Pasted below if requested]

Thank you!
```

## Support Best Practices

### For Fastest Resolution

**Do:**
- ✅ Check FAQ first
- ✅ Update to latest version
- ✅ Restart app before reporting
- ✅ Include screenshots
- ✅ Describe steps to reproduce
- ✅ Be specific and detailed
- ✅ Be patient and polite

**Don't:**
- ❌ Send multiple emails for same issue
- ❌ Use aggressive language
- ❌ Expect instant replies
- ❌ Ask for business advice
- ❌ Request illegal assistance

### Communication Guidelines

**Effective Support Requests:**

**Good Example:**
```
Subject: ThriftFlip Support - Barcode Scanner Not Working

Issue: When I tap "Scan Barcode," the camera opens but 
doesn't detect any barcodes. I've tried 5 different products.

iOS Version: 18.2
ThriftFlip Version: 2.0
Device: iPhone 15 Pro

Steps:
1. Open ThriftFlip
2. Tap "Scan Barcode"
3. Camera opens
4. Point at barcode
5. Nothing happens (should auto-detect)

I've tried:
- Restarting app
- Checking camera permissions (enabled)
- Different lighting
- Different barcodes

Screenshot attached.

Thank you!
```

**Poor Example:**
```
Subject: help

App doesn't work. Fix it.
```
