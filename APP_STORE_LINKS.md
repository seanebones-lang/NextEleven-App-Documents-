## App Store Connect Configuration

### Step 1: Add EULA Link

**Option A: Custom EULA (Recommended)**
1. Go to App Store Connect → Your App → App Information
2. Scroll to "End User License Agreement (EULA)"
3. Select "Use Custom EULA"
4. Copy and paste content from `EULA.md`

**Option B: Standard Apple EULA with Link**
1. Keep "Use Standard Apple EULA" selected
2. Add this link to your App Description:
   ```
   Terms of Use: https://github.com/seanebones-lang/NextEleven-App-Documents-/blob/main/EULA.md
   ```

### Step 2: Add Privacy Policy Link

1. Go to App Store Connect → Your App → App Information
2. In "Privacy Policy URL" field, enter:
   ```
   https://raw.githubusercontent.com/seanebones-lang/NextEleven-App-Documents-/main/Privacy_Policy.md
   ```

### Step 3: Update App Description

Add these links to your App Description section:

```
Terms of Use: https://github.com/seanebones-lang/NextEleven-App-Documents-/blob/main/EULA.md

Privacy Policy: https://github.com/seanebones-lang/NextEleven-App-Documents-/blob/main/Privacy_Policy.md

Subscription Terms: https://github.com/seanebones-lang/NextEleven-App-Documents-/blob/main/Subscription_Terms.md
```
