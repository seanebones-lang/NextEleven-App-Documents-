## General Questions

### What is Zombie Parrot?

Zombie Parrot is an AI-powered coding assistant for iOS developers. It helps you generate, debug, and refactor Swift code using the Grok Code Fast 1 AI model from xAI.

### What can Zombie Parrot do?

- Generate Swift code from natural language descriptions
- Debug and fix code issues
- Refactor code for better structure
- Provide conversational AI assistance
- Accept voice input for hands-free interaction

### What devices are supported?

- **iOS:** iPhone and iPad running iOS 19.0 or later
- **Device Models:** All devices that support iOS 19.0+
- **Storage:** Minimum 50MB free space

### Is Zombie Parrot free?

Zombie Parrot offers a **free tier** with:
- Basic code generation
- Basic debugging
- 10 queries per day
- Community support

**Pro subscription** unlocks:
- Unlimited queries
- Agent Swarm
- Code refactoring
- Priority support
- And more

## Getting Started

### How do I get started?

1. Download the app from the App Store
2. Open the app
3. Set up your API key (Settings > API Configuration)
4. Start generating code!

See **Settings > Onboarding** for a detailed guide.

### Do I need an API key?

Yes, you need an xAI API key to use Zombie Parrot. You can:
- Use the default API key (if available)
- Get your own key from https://console.x.ai

### How do I get an xAI API key?

1. Visit https://console.x.ai
2. Sign up or log in
3. Go to API Keys section
4. Generate a new key
5. Copy the key (starts with "xai-")
6. Enter it in Zombie Parrot Settings

### Is my API key secure?

Yes! Your API key is stored securely in Apple's Keychain with hardware-backed encryption. We never transmit, store, or access your API keys on our servers.

## Features

### What is Agent Swarm?

Agent Swarm is a Pro feature that uses multiple AI agents working in parallel:
- **Generator Agent:** Creates new code
- **Optimizer Agent:** Improves performance
- **Tester Agent:** Generates unit tests

You get combined results from all agents for comprehensive solutions.

### What is code refactoring?

Code refactoring improves your code structure without changing functionality:
- Applies SOLID principles
- Improves code organization
- Enhances readability
- Maintains functionality

This is a **Pro feature**.

### Can I use voice input?

Yes! Zombie Parrot supports voice input:
1. Select **Voice** mode
2. Grant microphone permissions
3. Tap the microphone button
4. Speak your request
5. Review and send

### How accurate is the generated code?

AI-generated code is generally accurate but should always be:
- Reviewed before use
- Tested thoroughly
- Validated for your specific use case

The AI uses Grok Code Fast 1, which is specifically trained for coding tasks.

## Technical Questions

### Why do I need an API key?

Zombie Parrot uses the xAI Grok API to generate code. The API key authenticates your requests and allows you to use the service. Your key is stored securely and never shared.

### What if I hit the query limit?

Free tier users are limited to 10 queries per day. To get unlimited queries:
- Upgrade to Pro subscription
- Wait until the next day (limit resets daily)

### Why is my code generation slow?

Code generation speed depends on:
- Internet connection speed
- xAI API response time
- Complexity of your request
- Current API load

Typical response time: 2-5 seconds.

### Can I use Zombie Parrot offline?

Partially. You can:
- View cached results when offline
- Access previously generated code
- But cannot make new API requests

Pro subscribers get enhanced offline caching.

### What AI model does Zombie Parrot use?

Zombie Parrot uses **Grok Code Fast 1** from xAI, which is specifically designed for coding applications with rapid reasoning for agentic coding tasks.

## Troubleshooting

### The app won't start. What should I do?

1. Restart your device
2. Update iOS (need iOS 19.0+)
3. Reinstall the app
4. Check device storage
5. Contact support: info@mothership-ai.com

### I'm getting "Invalid API Key" errors.

1. Verify your key starts with "xai-" or "gsk-"
2. Check for extra spaces or characters
3. Verify your key is active at https://console.x.ai
4. Try generating a new key
5. Update the key in Settings

### Why am I getting network errors?

1. Check your internet connection
2. Verify WiFi or cellular data is enabled
3. Check if api.x.ai is accessible
4. Try using cached results if offline
5. Check firewall/VPN settings

### My subscription isn't working.

1. Go to Settings > Restore Purchases
2. Verify subscription in iOS Settings > Subscriptions
3. Ensure you're signed in with correct Apple ID
4. Contact support: info@mothership-ai.com

### Voice input isn't working.

1. Grant microphone permissions (iOS Settings > Zombie Parrot)
2. Grant speech recognition permissions
3. Check microphone isn't blocked
4. Reduce background noise
5. Ensure good internet connection

## Privacy and Security

### What data do you collect?

We collect minimal data:
- Device information (for analytics)
- Usage statistics (to improve the app)
- Crash reports (to fix bugs)
- API keys (stored securely in Keychain)

See **Settings > Privacy Policy** for complete details.

### Do you sell my data?

**No.** We never sell your personal information to third parties.

### Where is my data stored?

- **API Keys:** Stored in Apple Keychain (device only)
- **Conversations:** Stored locally on your device
- **Analytics:** Stored securely, anonymized after 90 days

### Can I delete my data?

Yes! You can:
- Delete conversation history (Settings > Clear History)
- Remove API keys (Settings > Remove API Key)
- Disable analytics (Settings > Privacy)

### Is my code shared with others?

No. Your code and queries are:
- Sent only to xAI for processing
- Not shared with other users
- Not used to train models (subject to xAI's policies)
- Stored locally on your device
