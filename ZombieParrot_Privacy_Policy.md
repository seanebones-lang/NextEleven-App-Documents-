## 1. Information We Collect

### 1.1 Information You Provide

- You may provide xAI API keys to access Grok AI services

**Network Information**

### 1.3 Information from Third Parties

- Purchase history and subscription status

**xAI Grok API**

## 4. Data Storage and Security

### 4.1 Local Storage

**On-Device Storage**
- Conversations stored locally using UserDefaults (encrypted)
- API keys stored in Apple Keychain (hardware-backed encryption)
- Settings and preferences stored locally
- Cache entries stored locally (50 entry limit)

**iCloud Storage (Optional)**
- If enabled, data syncs across your Apple devices
- Encrypted using Apple's iCloud security
- You can disable iCloud sync in Settings
- Data is encrypted in transit and at rest

### 4.2 Security Measures

We implement industry-standard security measures:
- **Encryption:** API keys stored in Keychain with hardware-backed encryption
- **HTTPS:** All network communications use TLS 1.2+
- **Code Security:** Regular security audits and updates
- **Access Controls:** Minimal data collection principle
- **Secure Coding:** Following OWASP mobile security guidelines
- **Regular Updates:** Security patches applied promptly

### 4.3 Data Retention

- **Conversation History:** Stored until you delete it or uninstall the app
- **Usage Analytics:** Retained for 90 days, then anonymized
- **Crash Reports:** Retained for 1 year for debugging purposes
- **Subscription Data:** Retained as required by Apple and tax laws (7 years)
- **API Keys:** Stored until you delete them or uninstall the app

## 5. Your Rights and Choices

### 5.1 Access and Control

You have the right to:
- **Access:** View all data stored in the app
- **Delete:** Clear conversation history, delete account data
- **Export:** Export your data (contact us for assistance)
- **Opt-out:** Disable analytics collection in Settings
- **Correction:** Correct inaccurate data

### 5.2 Managing Your Data

**Delete Conversation History**
- Go to Settings > Clear History
- All local conversations will be deleted immediately

**Delete API Keys**
- Go to Settings > Remove API Key
- Keys will be removed from Keychain immediately

**Disable Analytics**
- Go to Settings > Privacy > Disable Analytics
- Analytics collection will stop immediately

**Export Your Data**
- Contact us at info@mothership-ai.com
- We will provide your data in a machine-readable format within 30 days

### 5.3 California Privacy Rights (CCPA)

If you are a California resident, you have additional rights:

- Right to non-discrimination for exercising your rights
- Right to data portability

To exercise these rights, contact us at info@mothership-ai.com.

### 5.4 European Privacy Rights (GDPR)
