## Our Privacy Commitment

Ultimate Life Coach is built with **privacy-first principles**. We believe your personal data, conversations, and emotional insights should remain private and under your control.

## Information We Collect

### 1. Information You Provide

- **Conversation History:** All chat messages are stored locally on your device
- **Mood Entries:** Your mood check-ins and notes are stored locally
- **Starred Insights:** Insights you save are stored locally
- **API Keys:** If you choose to use your own xAI API key, it is stored securely in your device's Keychain

### 2. Information Collected Automatically

- **Subscription Status:** Apple provides subscription status via StoreKit (required for app functionality)
- **App Usage:** Basic app functionality data (no personal identifiers)

### 3. Information We Do NOT Collect

We do **NOT** collect:
- ❌ Your name, email, or personal identifiers
- ❌ Location data
- ❌ Device identifiers for tracking
- ❌ Analytics or tracking data
- ❌ Third-party analytics
- ❌ Advertising identifiers
- ❌ Cross-app tracking data

## How We Use Your Information

### Local Storage (On Your Device)

- **Conversations:** Stored locally in JSON format for app functionality
- **Mood Data:** Stored locally for mood tracking features
- **Insights:** Stored locally for your insight journal
- **Settings:** Stored locally for app preferences

### API Usage

- **xAI Grok API:** When you send messages, they are sent to xAI's Grok 4.1 Fast Reasoning API for processing
- **Photos:** If you share photos, they are sent to xAI's API for analysis (not stored by us)
- **Default API Key:** The app includes a default API key for convenience. You can use your own key in Settings
- **Custom API Key:** If you provide your own API key, it is stored securely in your device's Keychain and never shared with us

### Subscription Information

- **Apple StoreKit:** Subscription status is managed by Apple and required for app access
- **No Personal Data:** We do not receive your payment information or personal details from Apple

## Data Storage and Security

### Local Storage

- All data is stored **100% locally** on your device
- Data is stored in your device's Documents directory
- Data is **not** synced to iCloud by default
- Data is **not** backed up to our servers
- Data is **not** accessible to us

### API Key Security

- Custom API keys are stored in iOS Keychain (encrypted)
- Default API key is embedded in the app (for convenience)
- API keys are never transmitted to our servers
- API keys are only used for xAI API calls

### Data Encryption

- Local data files are stored in plain JSON format
- iOS provides file system encryption
- Keychain items are encrypted by iOS
- All API communications use HTTPS/TLS

## Third-Party Services

### xAI Grok API

- **Purpose:** AI-powered conversation processing
- **Data Sent:** Your messages and optional photos
- **Data Received:** AI-generated responses
- **Privacy:** Subject to xAI's privacy policy
- **Location:** xAI servers (United States)
- **Retention:** We do not control xAI's data retention

**xAI Privacy Policy:** https://x.ai/privacy

### Apple Services

- **StoreKit:** Subscription management (required by Apple)
- **iCloud:** Not used by default (optional user choice)
- **Apple ID:** Not accessed by our app

## Your Rights and Choices

### Access Your Data

- All your data is stored locally on your device
- You can export chat history from Settings
- You can view all mood entries in the app
- You can access starred insights in the app

### Delete Your Data

- **Delete All Data:** Available in Settings → Delete All Data
- This permanently removes:
  - All conversation history
  - All mood entries
  - All starred insights
  - All app data
- **Warning:** This action cannot be undone

### Export Your Data

- **Export Chat History:** Available in Settings
- Exports as plain text
- You can share or save anywhere you choose

### API Key Management

- **Use Default Key:** App includes a default API key
- **Use Your Own Key:** Available in Settings → API Configuration
- **Remove Custom Key:** Available in Settings
- **Switch Anytime:** You can switch between default and custom keys
